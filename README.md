# Team 15 - Environmental Justice

# This project aims to analyze how platforms are used to discuss and mobilize environmental justice, and the correlation of these online trends with actual environmental inequalities. Using the Kaggle dataset “Global Climate Action Social Media Trends 2024-25”, we’ll track metrics on posts, hashtags, sentiment, and geography of climate-related content. Aiming to identify patterns in people “digital activism” and compare the alignment or divergence from actual climate disparities.
